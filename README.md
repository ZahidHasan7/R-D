# Bangla Voice Agent Demo

This is a production-ready Bangla Voice Agent with neural TTS (VITS2) and ASR (Whisper).

## Key Features
- **Neural TTS:** High-fidelity Bangla speech synthesis using Meta MMS.
- **Improved Normalizer:** Advanced text preprocessing for abbreviations and units.
- **Optimized Pipeline:** Real-time inference configured for CPU environments.

## Deployment Info
This Space is built using Docker.
- **Port:** 7860
- **Base Image:** python:3.10-slim
